# 🌦 Advanced Weather Dashboard (Streamlit App)

An interactive and modern Weather Web Application built using **Streamlit** that displays real-time weather data, 5-day forecasts, auto-detected location, weather icons, animated UI, and temperature charts.

---

## 🚀 Features

- 🌍 Auto-detect user location
- 🌡 Current temperature display
- 💧 Humidity & Wind Speed
- 🌥 Weather condition with icons
- 📅 5-Day forecast
- 📈 Temperature trend chart
- 🌈 Animated gradient background
- 💎 Glassmorphism modern UI
- 🔐 Secure API key using `.env`

---

## 🛠 Tech Stack

- Python
- Streamlit
- OpenWeatherMap API
- Requests
- Pandas
- Matplotlib
- Python-dotenv

## 🔐 Setup Environment Variable

Create a `.env` file in the project root:

---

## ▶ Run Locally
OPENWEATHER_API_KEY=your_api_key_here


⚠ Do NOT upload `.env` to GitHub.

Add this to `.gitignore`:


.env


---

## ▶ Run Locally

1. Clone the repository:


git clone https://github.com/your-username/your-repo-name.git

cd your-repo-name


2. Install dependencies:


pip install -r requirements.txt


3. Run the app:


streamlit run app.py


---

## ☁ Deployment (Streamlit Cloud)

1. Push project to GitHub
2. Go to Streamlit Cloud
3. Connect repository
4. Add API key in:
   Settings → Secrets

Example:


OPENWEATHER_API_KEY = "your_api_key_here"


Deploy and done ✅
